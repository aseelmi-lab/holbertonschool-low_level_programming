# C - Recursion

## Description
This project focuses on understanding and implementing **Recursion** in C programming. Recursion is a method where the solution to a problem depends on solutions to smaller instances of the same problem.

## Learning Objectives
* What is recursion and how to implement it.
* In what situations you should (and shouldn't) implement recursion.
* Understanding the Base Case and the Recursive Step.

## Requirements
* Allowed editors: vi, vim, emacs.
* Compiled on Ubuntu 20.04 LTS using gcc.
* All files end with a new line.
* No loops or static variables allowed.

## Tasks
* **0-puts_recursion.c**: Prints a string followed by a new line.
